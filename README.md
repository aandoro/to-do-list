# Challenge Ensolvers

## Pre-requirements

- nodejs 16.13
- npm 8.1

global install "forever" for run commands bash

```bash
npm install forever -g
```

## Technologies

- Nestjs 8.0
- Mongo 5.0
- Mongoose 6.1
- Vuejs 2.6

## Install & Run

first run

```bash
./script.sh install
```

after run

```bash
./script.sh start
```

if you want stop the apps run

```bash
./script.sh stop
```
